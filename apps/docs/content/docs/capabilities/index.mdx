Capabilities are optional tools that Mate can configure during `mate setup` and sync into linked working repositories when you run `mate opencode` or `mate claude`.

For now, this page only lists the projects Mate can connect.

## Projects

- [OpenSpec](https://openspec.dev/) for spec-driven change workflows
- [Graphify](https://github.com/safishamsi/graphify) for repository knowledge graphs
- [TokenSave](https://github.com/aovestdipaperino/tokensave) for code-graph exploration through MCP
- [Headroom](https://github.com/chopratejas/headroom) for AI workflow support
- [React Doctor](https://www.react.doctor/) for React diagnostics

## How They Fit

Run `mate setup` in the companion repository, choose the capabilities you want, then link the working repository with `mate repo link`.

After that, `mate opencode` and `mate claude` launch from the working repository with the selected companion-managed capabilities available.
